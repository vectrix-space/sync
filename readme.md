Provides thread-safe utilities for highly concurrent scenarios in Java 17+.

* [Sync Collections](collections/readme.md)
* [Sync Collections `fastutil`](collections-fastutil/readme.md)

## Building

__Note:__ If you do not have [Gradle](https://www.gradle.org/) installed then use `./gradlew` for Unix systems or Git Bash and gradlew.bat for Windows systems in place of any 'gradle' command.

In order to build Sync you simply need to run the `gradle build` command.
